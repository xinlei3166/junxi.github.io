开源项目 ​

业余时间开发的一些开源项目，主要基于Web前端和Node.js技术栈。

Vue ​

  • vue-virtual-components - Vue3 虚拟组件

  • vite-vue-template - Vue 模板, Vite5 + Vue3 + Pinia + Vue Router + TypeScript + Ant Design Vue

  • vite-vue-mobile-template - Vue 移动端模板, 使用 Vite5 + Vue3 + Pinia + Vue Router + TypeScript + Vant

  • vite-uniapp-template - Uniapp 模板, 使用 Vite5 + Vue3 + Pinia + TypeScript + UnoCSS + NutUI + Uniapp

  • vite-vue2-template - Vue2 模板, 使用 Vite4 + Vue2 + Pinia + Vue Router + TypeScript + Ant Design Vue

  • vite-electron-template - Vite4 + Vue3 + Electron24

  • taro-vue-template - Taro 模板, 使用 Webpack5 + Vue3 + Pinia + TypeScript + UnoCSS + NutUI + Taro

  • ts-template - typescript 项目模版

  • ts-monorepo-template - typescript monorepo 项目模版

  • vueui-template - vue组件库项目架子, rollup 打包js, gulp打包css

  • iview-admin-template - iview-admin-template is built using typescript and iviewui

  • iview-filter-table - 一个基于iView Table 的带搜索过滤的Table组件, 支持 Input输入框、Select下拉框、DatePicker时间框三种表格筛选方式

React ​

  • vite-react-template - Vite4 + React18 + Redux4 + React-Router6 + Unocss + Typescript + Ant-Design

  • umi-react-template - Umi + React + Unocss

Vuepress / Vitepress ​

  • vitepress-theme-demoblock - 一个基于 Vitepress 的主题插件，它可以帮助你在编写文档的时候增加 Vue 示例。

  • vitepress-theme-website - 一个基于 Vitepress 的主题插件，集成评论Waline、Live2D看板娘。

  • vuepress-plugin-demoblock-plus - 一个基于 Vuepress 2.x 的插件，它可以帮助你在编写文档的时候增加 Vue 示例。

  • vuepress-plugin-waline - 一个基于 Vuepress 2.x 的插件，它可以帮助你更快的接入 Waline 评论系统。

  • vuepress-plugin-live2d-plus - 一个基于 Vuepress 2.x 的插件，它可以帮助你集成Live2D看板娘。

Utilities / CLI ​

  • @yunquejs/release - npm package version management，include build、test、push github、npm publish

  • @yunquejs/cli - Simple scaffolding tools

  • xlogger-pretty - A logger as pretty and easy to use as the Python logging library

  • exception-error - A small error handling library like python exception handling that supports extensions

  • video-tools - 用 electron 和 ffmpeg 做的一个视频转换工具，m3u8转换为mp4并下载到本地

  • xlform - 一个仿 django form 的表单验证库

  • xlmongo - Mongodb连接池和SQL语句封装

Python ​

  • xlup - 一个使用sanic、typescript和vue实现的图片视频上传服务系统

  • xlform - 一个仿 django form 的表单验证库

  • xlcmdb - 初学python和前端知识，利用Bootstrap + saltstack + paramiko + python 做了一个简单的CMDB平台
